LIRI Node Appclose

LIRI is command line node.js application that takes in parameters and gives you back data.

Similar to iPhone's SIRI, which is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface.

There are three categories of data you can pull from LIRI: song data from Spotify's API, movie data from the OMDB API, and concert data from the Bands in Town API.

Bamazonmore_vert

GitHub Repo

Bamazon Node Appclose

Bamazon is a node.js application that utilizes mySQL and the node package "Inquirer" to create an Amazon-like storefront.

There are two different modes, customer and manager. As a customer, the app takes in orders and then depletes stock from the store's inventory stored in a mySQL database. As a manager, the app allows you to add new product to the store as well as replenish inventory.
